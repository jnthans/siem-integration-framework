# Changelog

## v1.0.0 — 2026-03-28

### Initial release

Framework distilled from three production Wazuh integrations:
- [wazuh-1password](https://github.com/jnthans/wazuh-1password)
- [wazuh-proofpoint](https://github.com/jnthans/wazuh-proofpoint)
- [wazuh-cortex-xdr](https://github.com/jnthans/wazuh-cortex-xdr)

Includes:
- Complete architecture documentation (4-layer design, data flow, design principles)
- Five-phase build process (planning, building, testing, deploying, documenting)
- Guides for AI-assisted development, SIEM adaptation, and security review
- Reference docs for repo structure, coding conventions, and rule design
- Scaffold templates for all integration files
- Five LLM reference files for machine-readable context
- GitHub Pages documentation site
